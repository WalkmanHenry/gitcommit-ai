# GitCommit AI

🤖 AI-powered Git commit message generator for VS Code

自动生成规范的 Git 提交信息，支持 Conventional Commits 规范。

## ✨ 特性

- 🎯 **智能分析** - 根据代码变更自动生成提交信息
- 📝 **多种风格** - 支持 Conventional / Simple / Detailed 三种风格
- 🏷️ **类型选择** - 支持手动选择提交类型 (feat/fix/docs等)
- ⚡ **快速生成** - 一键生成，无需配置
- 🔒 **本地运行** - 无需联网，保护隐私

## 🚀 使用方法

1. 在 VS Code 中安装插件
2. 修改代码并保存
3. 打开 Source Control 面板
4. 点击 sparkle 图标或运行命令 `Generate Commit Message`
5. 提交信息将自动填入输入框
